Forward pass of the layer. Forward is a core function of any layer, so let's do a simple example:

Suppose we have the following layer: 
import torch
import torch.nn as nn

class Layer(nn.Module):
  # constructor
  def __init__(self):
    super(Layer, self).__init__()
    self.weight = 2;
        
  def forward(self, x):
    return x * self.weight;
    # dead simple
    # return the product of weight and input x

Then we instatiate an instance of the layer and pass a simple input through it with forward(): 
simple = Layer()
x = 3
simple(3)

Output: 6
